使用方法

将鼠标光标置于技能名称后，会显示技能的简易说明。

单击[保存地址和技能列表]按钮后会显示对话框。

  • ・ 以URL形式顯示獲取的技能一覽表和技能槽狀態
  • ・ 以文本形式顯示获得的技能一览

雙擊 D&D 或技能圖標可以把技能放置在技能槽内 。

  • ・ 雙擊技能槽的技能圖標可以取消顯示
  • ・ 如果不能变更技能槽内的技能位置，请解除——→重新配置
